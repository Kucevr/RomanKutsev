# Roman Kutsev - Portfolio
## Design Concept
- **Aesthetic:** Brutalist Editorial / Minimalist Typographic.
- **Vibe:** Strict lines, high contrast, immense typography dominating the viewport, delicate serifs interrupting rigid mono-spaced fonts.
- **Color Palette:** Off-white (`#E5E4DE`) with sheer stark black (`#111`) for maximum contrast without absolute white harshness.

## Tech Stack
- Frontend: **React + Vite**
- Styling: **Tailwind CSS 3.x/4.x**
- Animation: **GSAP** (GreenSock) for high-performance timeline orchestration and staggered reveals.
- Typography: 
  - *Oswald* for massive tracking-tighter headlines.
  - *Instrument Serif* for italic, luxurious contrast.
  - *JetBrains Mono* for UI elements, raw code-like aesthetic, matching the "ENGINEER" identity.

## Features
- **GSAP Orchestrated Intro:** Carefully timed entrance for each element - text fades and slides from bottom to top, revealing the main layout.
- **Typography as Interface:** The layout relies very little on boxes and entirely on text scaling. Using vw units to keep the design fluid and massive across any screen.
- **Hover Micro-Interactions:** Subtle background transitions, image scalings, and classic brutalist strike-throughs on hover.
